# 🚀 ÉDIT MARKET PRO

## 🏷️ Marketplace Premium de Produits Numériques

ÉDIT MARKET PRO est une plateforme e-commerce moderne dédiée aux créateurs de contenu :
- Montage vidéo
- Motion design
- VFX & assets
- Audio & SFX
- Templates professionnels
- Formations créatives

---

## 🎯 OBJECTIF DU PROJET

Créer une marketplace **100% frontend (sans backend)** utilisant :

- HTML
- CSS (Glassmorphism premium)
- JavaScript
- JSON (base de données locale)
- localStorage (auth & session)
